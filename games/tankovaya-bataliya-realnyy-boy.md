# Танковая баталия, реальный бой

- Автор: Сергей Храмов
- Источник: PDF "Танковая баталия", редакция 2016
- Программа: [tankovaya-bataliya-realnyy-boy.txt](tankovaya-bataliya-realnyy-boy.txt)

## Описание

Вариант игры для двух участников на одном ПМК. Игроки управляют танками, расходуют топливо и мины, задают угол и скорость выстрела. В варианте "реальный бой" игрок перед выстрелом не знает текущий коэффициент погоды.

### Подготовка

Рекомендуемые регистры из источника:

- `R0 = 9,81`
- `RD = 50`
- `RE = 50`
- `R1 = 20`, `R2 = 20`
- `R4 = 5`, `R5 = 5`
- `R7 = 4000`, `R8 = 4000`
- `R3` любое число
- `R6 = 0`
- `RB = 250`

Переключатель `Р-Г` поставьте в градусы.

### Запуск

Нажмите `БП A4 С/П`. Затем введите `1` для хода танка N1 или `2` для хода танка N2. Для выстрела введите `угол`, `↑`, `скорость`, `С/П`.
