## Описание

Короткая генераторная программа для лотерейных комбинаций. Перед запуском задается верхняя граница диапазона, после чего программа выдает очередные числа по остановкам `С/П`.

## Запуск

После ввода программы:

1. Перейти в автоматический режим: `F АВТ`.
2. Записать верхнюю границу лотереи в `R7`: `36 хП7`, `45 хП7` или `49 хП7`.
3. Инициализировать случайное состояние: `КСЧ хП9`.
4. Запустить: `В/О С/П`.
5. Для следующего числа нажимать `С/П`.

В исходной таблице закрашенная клетка после `F АВТ` подписана как `36 или 45 -> хП7`; это инструкция запуска, а не адрес программы.

## Примеры из тетради

- `6 из 45` -> `23, 1, 38, 19, 5, 25`
- `5 из 36` -> `7, 3, 34, 17, 15`
- `6 из 49` -> `45, 2, 37, 11, 25, 22`
- `No TOB` -> `63, 41, 94, 18, 7, 22, 25, 31, 57`
