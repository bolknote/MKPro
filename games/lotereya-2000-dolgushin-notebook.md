## Описание

Лотерея `5 из 36`. Игрок загадывает пять чисел от `1` до `36`, платит за каждое число и получает выигрыш по количеству совпадений.

## Подготовка

Перейти в `F АВТ` и ввести начальные деньги в регистр `d`; в тетради приведён пример `50`:

```text
50 хПd
```

Начало игры:

```text
КСЧ
В/О
С/П
```

На экране появится `0`.

## Правила

Можно загадывать числа от `1` до `36` включительно.

Ввести первое загаданное число и нажать `С/П`, затем второе число и `С/П`, и так до пятого числа. Через две с половиной минуты на экране появится количество угаданных чисел.

Нажать `↔`, чтобы увидеть выигрыш. `Пхd` показывает ваши деньги.

Если на экране цифра `5`, значит сорван джекпот: `22026`.

Чтобы узнать выигрышные числа:

```text
Пх1 ; первое число
Пх2 ; второе число
Пх3 ; третье число
Пх4 ; четвёртое число
Пх5 ; пятое число
```

Чтобы продолжить, нажать `С/П`. На экране снова появится `0`.

Если игроку не везёт и заканчиваются все деньги, на экране появляется символ `CEL 96.`. За одно число забирается `1` гривна, то есть пять чисел стоят `5` гривен.

## Таблица выигрышей

| Угадано чисел | Выигрыш |
| --- | --- |
| `0` | `0` грв. |
| `1` | `7` грв. |
| `2` | `54` грв. |
| `3` | `403` грв. |
| `4` | `2980` грв. |
| `5` | `22026` грв. |

## Коды изменения игры

В тетради приведены такие правки:

```text
26 4
27 5
```

Лотерея станет `5 из 45`.

```text
08 N
```

Стоимость одного числа станет `N`, где `N < 10` гривен.

```text
74 Fe^x
```

Повысится таблица выигрышей.

```text
87 ВП
88 .
```

Символ `CEL 96.` изменится на `ЕГГОГ`.
