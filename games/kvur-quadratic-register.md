# Quadratic Equation Solver with Registers (MK61/52)

- Source: [Arbinada — КВУР - квадратное уравнение ...](https://pmk.arbinada.com/category/01167.html)
- Program: [kvur-quadratic-register.txt](kvur-quadratic-register.txt)

## Description
Solves quadratic equations `a*x^2 + b*x + c = 0` using MK-61/52 register-based implementation.

Input coefficients are supplied in the register layout described in the article; the program returns real or complex roots according to the discriminant.
