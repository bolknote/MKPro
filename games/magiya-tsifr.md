# Магия цифр

- Автор: Владимир Хмелюк
- Источник: [Lord_BSS](https://lordbss.narod.ru/pmk109.html)
- Программа: [magiya-tsifr.txt](magiya-tsifr.txt)

## Описание

Наберите дату рождения исследуемого (год месяц число), В/О С/П...

RX="N.M", где N - цифра-код, М - число повторений этой цифры в проверяемых числах судьбы.

**Коды:**

- самолюбие

- общительность

- доброта

- везучесть

- хитрость

- талант

- принципиальность

- отзывчивость

- трудолюбие

Записывайте результат и нажимайте С/П, пока на индикаторе не высветится "ВСЕ" - теперь можно исследовать следующего человека. Ввод: его дата рождения, С/П...

### Пример
- 19500101 (1 января 1950 года) В/О С/П.

- RX="1.5" - очень развитое самолюбие; С/П.

- RX="2.0" - нулевая общительность, нелюдимость; С/П.

- RX="3.0" - хорошо выраженная злость; С/П.

- RX="4.1" - не ахти какая везучесть; С/П.

- RX="5.3" - хитрец; С/П.

- RX="6.2" - есть немного таланта; С/П.

- RX="7.1" - не очень принципиальный тип; С/П.

- RX="8.0" - сколько не аукайся, не откликнется; С/П.

- RX="9.1" - недалеко до лоботряса; С/П.

- RX="ВСЕ" - следующий!
