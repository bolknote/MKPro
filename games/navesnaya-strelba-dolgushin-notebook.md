## Описание

Артиллерийская игра: игрок управляет орудием и задает угол возвышения, пытаясь попасть по вражеской пушке навесной траекторией.

На поле боя:

- `8` - ваше орудие
- `-` - вражеская пушка
- `E` - взорвавшееся ядро
- `1` - ядро взорвалось рядом с врагом

## Подготовка

1. Переключатель поставить в положение `Г`.
2. Режим `F АВТ`.
3. Ввести:

```text
37 хПd
0 хП4
1000 хП1
80005 КИНВ хПЕ
8888 0008 В↑ 86420002 K∨ K{x} ВП 2 хПC
```

4. Ввести параметры:

```text
расстояние до цели, м -> хП7
максимальная дальность стрельбы пушки, м -> хП8
радиус поражения взрывом ядра, м -> хП9
```

В тетради приведены примеры: `200` для расстояния и `10` для радиуса поражения.

## Игра

1. Выполнить `В/О`, `С/П`; на экране появится поле боя.
2. Ввести угол возвышения орудия и нажать `С/П`.
3. После выстрела программа показывает поле боя с местом взрыва.
4. Чтобы узнать, в скольких метрах от врага взорвалось ядро, выполнить `Пх5`; отрицательное число означает, что ядро не долетело.
5. После попадания по врагу на экране выводится результат вида `,8,C-115-`, где число показывает количество набранных очков. Чем меньше затрачено снарядов, тем больше очков.
