# Бастион

- Автор: Владимир Сивицкий
- Источник: *Пионерия/Одноклассник*, №11, 1990, mirrored at [arbinada.ddns.net](http://arbinada.ddns.net/pmk-files/magazines/%D0%9F%D0%B8%D0%BE%D0%BD%D0%B5%D1%80%D0%B8%D1%8F%201990-91%20%D0%9F%D0%9C%D0%9A.zip)
- Программа: [bastion.txt](bastion.txt)

## Описание

Игра для тренировки глазомера. Игрок вводит угол стрельбы по бастиону, а калькулятор вычисляет результат выстрела.

Перед началом занесите константы: `360` в `R7`, `11.11111` в `RA`. Затем запустите программу с `В/О` `С/П`.
