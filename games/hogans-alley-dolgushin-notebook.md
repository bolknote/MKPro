## Описание

Стрелковая игра по мотивам `Hogan's Alley`: на экране появляются невинные люди и бандиты. Нужно стрелять только по бандитам и не попадать в гражданских.

## Подготовка

В режиме `F АВТ` ввести в регистры:

```text
7070707 хПB
100 хПE
40 хПd
801 хП1
0 хП4         ; сумматор очков
80001 хП2
40 хП0
8000001 хП3
8888888 В↑ 86222223 K∨ K{x} ВП 7 хПC
8888888 В↑ 87777771 K∨ хП5
8888808 В↑ 87777713 K∨ хП6
8888088 В↑ 87777133 K∨ хП7
8880888 В↑ 87771332 K∨ хП8
88808888 В↑ 87713322 K∨ хП9
```

После ввода нажать `В/О`, `С/П`.

## Обозначения

| Символ | Значение | Действие |
| --- | --- | --- |
| `E` | `LADY` | убивать нельзя |
| `C` | `POLICE` | убивать нельзя |
| `.` | `BABY` | убивать нельзя |
| `8` | `PROFFESSOR` | убивать нельзя |
| `Б` | `OLD WOMEN` | убивать нельзя |
| `Г` | `GANG A`, `GUNMAN` | цель |
| `L` | `GANG B`, `KILLER` | цель |
| `9` | `GANG C`, `MAFIA` | цель |
| `7` | `GANG D`, `PUNK` | цель |

За уничтожение бандита прибавляется `1` очко. За уничтожение невинных людей забирается `3` очка.

## Управление

Переключатель `Р-ГРД-Г` задаёт направление выстрела.

| Положение переключателя | Действие |
| --- | --- |
| `Р` | стрелять влево |
| `ГРД` | стрелять в центр |
| `Г` | стрелять вправо |

Вид из глаз; экран движется вправо.

## Конец игры

Вы играете по Аллее Хогана и убиваете гангстеров. Если количество очков становится меньше `0`, игра заканчивается: `GAME OVER`, на экране `ЕГГ0Г`.

Если аллея успешно преодолена, на экране появляется `E----Э`. После этого нажать `↔`; будет показано количество набранных очков.

При попадании в невинных людей появляется `iLL`.

P.S. „Очистите улицу от ганстеров!“
