# Сумма кубов, 2-й способ

- Source: А. Г. Гайштут, `Калькулятор - твой помощник и соперник в играх`, 1988, с. 105.
- Program: [sum-cubes-formula-gaishtut-1988.txt](sum-cubes-formula-gaishtut-1988.txt)

## Description

Учебная программа вычисляет сумму кубов по формуле `(n(n+1)/2)^2`. В печатной колонке кода для адреса 05 указан `32`, но команда в книге и формула требуют `Fx^2`, то есть код `22`.

## Setup

После `F АВТ` введите `n` в `X`, затем `В/О С/П`.
