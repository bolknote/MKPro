## Описание

### N
- N это радиус поражения (3 - 9 км). Выставьте самостоятельно.

### Регистры
- R1-R3 - координаты целей, км (5-1000), т.е. дальность цели по отношению к границе (например, 25 хП1; 300 хП2; 826 хП3); R4 - чувствительность рулей, ° (5-30); Сх (кол-во израсходованных бомб) хП5; 0 (высота полёта самолёта, км) хПВ; 0 (угол наклона самолёта по отношению к горизонту, °) хПА; 0 (скорость полёта, км/ч) хПС; 0 (координата самолёта, км) хПД; 0 (зарплата лётчика) хПЕ; 55573155 Кинв K{x} ВП 7 хП6.

- **Р-ГРД-Г** ⇒ "Г".

Вы - пилот знаменитого штурмовика F-19. Ваша задача - уничтожить 3 цели на территории противника. Их координаты в R1-R3. После выполнения поставленной задачи нужно вернуться назад, т.е. снова пересечь границу.

**Старт:** БП 01.

**Управление:**

Нос на R4 ° вверх: 8 C/П.
Нос на R4 ° вниз: 2 C/П.
Не менять угол наклона: 5 C/П.
После этого RX="координата самолёта", RY="высота полёта самолёта".

Ускорение самолёта: БП 87 С/П.
Торможение самолёта: БП 91 С/П.
После этого RX="скорость F-19 (±10^-6 из-за влияния ветра)".

Сброс бомбы: 0 С/П. При попадании в цель RX="00", RY="координата взрыва". Если RX="X.1111111" - промах,
где X - кол-во уже сброшенных бомб. R8="координата взрыва".

После выполнения задачи для возвращения домой - угол наклона самолёта (RA) должен быть больше 90°.

RX="--ВСЕ--" - возврат домой. RY="Ваша зарплата, $".

RX="ЕГГОГ" - гибель штурмовика, если взрыв объекта произошёл ближе N км относительно самолёта, или скорость превысила 200 км/ч, и F-19 свалился в штопор.

### Примечание.
- Полёт при отрицательной скорости строго воспрещён!
