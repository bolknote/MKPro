Автор неизвестен.

Источник: тетрадь Рената Менькова "Мои игры МК-61 (Тетрадь 2)", `Рисунок (9).jpg`; `Рисунок (10).jpg`.

## Начальная подготовка

```text
10000000 K√ ВП хП2
000 K√ ВП ВП хП1
0 K√ ВП ВП хП0
25 хП3
В/О С/П
```

`В/0` в тетради читается как `В/О`, стандартная клавиша возврата к адресу `00`, а не цифра `1`.

## Управление и сигналы

`3` - сани находятся посередине дорожки.

`(-1)..1 ПП` - ввод приращения скорости. Практически это означает:

| Ввод | Значение |
| --- | --- |
| `1 ПП` | увеличить скорость |
| `0 ПП` | оставить скорость без изменения |
| `1 /-/ ПП` | уменьшить скорость |

Сигналы:

| Индикация | Значение |
| --- | --- |
| `Г00` | сани пересекли левую обочину |
| `E000000` | финиш |
| `Г` | неправильно набирают скорость |
| `Пх6` | время гонки |

## Примечания по чтению тетради

В первой строке `П6` читается как `хП6`, машинный код `46`.

`ПА` с кодом `4-` читается как `хПA`.

`ИПА` в формулах движения читается как `ПхA`, то есть извлечение из регистра `A`.

Закрашенный синим квадрат перед переходом к адресу `74` не является отдельной командой: вводится одна команда `Fx>=0`, затем адрес `74`.
