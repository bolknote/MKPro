# Easter Date Helper (Ascension and Pentecost)

- Source: [Arbinada — Вычисление даты Пасхи (61/52)](https://pmk.arbinada.com/category/00935.html)
- Program: [easter-date-passover-alt.txt](easter-date-passover-alt.txt)

## Description
This compact MK-61/52 routine derives the dates of Ascension and Pentecost using the results of the primary Easter calculation.

It is a small add-on that prepares follow-up liturgical dates from previously computed Easter values.
