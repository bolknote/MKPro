# Tekhnika Molodyzhi / KEI Collection

This folder contains source scans and articles from the «Клуб электронных игр»
and related features in *Tekhnika Molodyzhi*.

The files include listings from the 1985 issue section and the 1986–1987 KEI
column material.

This folder contains:

- `*.txt` — plain program listings, one command per addressed step.
- `*.md` — Markdown descriptions converted from the source pages.
- `manifest.tsv` — tab-separated index with program file, description file,
  title, author, and source reference.

Listings are transcribed from scans or OCR material and manually checked against
the printed tables.
