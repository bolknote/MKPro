- Source: [Tekhnika Molodyzhi, 1985 №7, «Мягкой посадки!»](https://epizodsspace.airbase.ru/bibl/tehnika_-_molodyoji/1985/7/put.html)
- Program: [soft-landing.txt](./soft-landing.txt)
- Also related to: [lunolet-1.txt](../semico/lunolet-1.txt)

## Description

This program is a variant from the **Elektronika B3-34 / Klub of Electronic Games** section in
`Техника-молодёжи` №7 (1985).
The issue introduces a modified landing scenario that is described as a variant of `Lunolot-1`.

The helper routines and initial input flow are reported as unchanged from the previous issue,
so the listing is kept compatible with the known `Lunolot-1` implementation.

The same issue also introduces the auxiliary **Planet Constructor (ПК-1)** program and the
`Guess Gravity` game concept built on top of this engine. Only the main landing code is
included as a full program listing in this repository.

## Setup

Start with `В/О С/П`.
