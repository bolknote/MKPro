# BUSSINESMAN

- Источник: тетрадь Александра Долгушина "Мои игры МК-61", программа 43, «BUSSINESMAN»
- Программа: [bussinesman-dolgushin-notebook.txt](bussinesman-dolgushin-notebook.txt)

## Описание

Настольная экономическая игра на поле с клетками призов, штрафов и банкротства. Игрок перемещает фишку по полю бросками кубика, меняет капитал и должен пройти пять кругов поля.

В источнике указано, что игра была написана в июне 2000 года и представляет собой настольную экономическую игру.

## Подготовка

Перед вводом поставить переключатель в положение `Р`.

В режиме `F АВТ` ввести:

```text
50 /-/ хП1
50 хП2
200 /-/ хП3
1000 хП4
41 хП7
8055955 KИНВ ВП 4 хП8
10000 хП9
831 KИНВ ВП 2 хПA
88008 В↑ 83056 K∨ K{x} ВП 4 хПC
1000 хПd
```

`1000 хПd` задаёт начальный капитал игрока.

Для начала нажать `В/О`, `С/П`.

## Ход игры

На экране показывается количество очков, выпавшее на кубике. Нажать `↔`, чтобы увидеть поле, на котором стоит фишка. В регистре `Р6` хранится номер поля, на котором в данный момент стоит фишка.

После вывода поля и просмотра регистров нажать `С/П`: будет сделан следующий бросок кубика.

Цель игры - пройти пять кругов поля.

## Обозначения

| Символ | Значение |
| --- | --- |
| `8--Б--` | банкротство, сгорают все очки |
| `8E` | штраф `50$` |
| `8Г` | премия `50$` |
| `8C` | штраф `200$` |
| `8L` | приз `1000$` |
| `8-` | пустое поле |
| `BCE` | игра пройдена, победа |
| `LOSE` | проигрыш, `GAME OVER` |

После появления символа `BCE` нажать `↔`: будет показано количество заработанных денег.

## Вариант поля

Если по адресу `27` вместо `Fsin` поставить `Fcos`, будет использоваться другое игровое поле.
