- Программа: [okhota-na-medvedya.txt](okhota-na-medvedya.txt)

## Описание

Охотник ищет медведя на координатной плоскости. Зверь слышен по направлениям, виден по горизонтали или вертикали, случайно перемещается и может напасть, если охотник подойдет слишком близко.

### Подготовка

Фиксированные значения из источника:

- `R0 = 1`
- `RC = 85555555`
- `RA = 85`
- `RD = 61`
- `RE = 34`

Рекомендуемые игровые регистры:

- `R2 = 10`
- `R3 = 3`
- `R4 = 0`
- `R5 = 6`
- `R9 = 5`
- `RB` - случайное число от 0 до 1
- `R6 = 5`, `R7 = -1` для начальных координат медведя `(-5; 1)`

### Запуск

Нажмите `БП 01 С/П`. После видеоизображения вводите перемещение охотника как `ΔX`, `↑`, `ΔY`, `С/П`. Если появляется `85555555`, охотник видит медведя и может стрелять.
