## Описание

Симулятор управления парусной яхтой. Игрок задает ветер, шаг времени и углы управления, а программа рассчитывает координаты яхты и общее время движения.

## Порядок игры

1. Установите угловой режим `градусы`.
2. Введите скорость ветра `Vв`, м/с, затем выполните `В/О С/П`. Результат `00` означает, что исходные данные приняты.
3. Введите шаг времени `Δt`, с, командой `хП0`.
4. Введите управление: угол `β`, затем `ПП`; угол `ψ`, затем `С/П`. Результат - координата `x`, м.
5. Нажмите `↔`, чтобы получить координату `y`, м. Нажмите `Пх6`, чтобы получить общее время `T`, с.
6. Продолжайте с пункта 3 или 4.

Если угол задан неверно, яхта теряет управление; сообщение `00` означает необходимость вернуться к пункту 4 с новыми `β` и `ψ`. Координаты остаются прежними. В начале `x0 = y0 = 0`.

Диаграммы источника задают допустимые диапазоны `β` относительно `ψ` по четвертям: `0 <= ψ <= 90`, `90 <= ψ <= 180`, `-180 <= ψ <= -90`, `-90 <= ψ <= 0`.

## Адаптация для МК-61

Оригинальный рукописный листинг подписан как МК-52, но не использует команды ППЗУ. Командная последовательность сохранена, запись команд приведена к формату МК-61.

Нижняя пометка источника `/0-6/ 50-56-56` по объяснению автора указывает занятые регистры (`R0`-`R6`) и размер программы: 50 фактически, 56 при округлении до кратности 7, 56 при округлении до кратности 8.
