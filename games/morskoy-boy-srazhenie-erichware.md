# Сражение

- Источник: [ErichWare, программы для МК-52](https://erichware.com/sovkalk/texty/promk52.htm), лист [52srajen.gif](https://erichware.com/sovkalk/texty/mk52/52srajen.gif)
- Программа: [morskoy-boy-srazhenie-erichware.txt](morskoy-boy-srazhenie-erichware.txt)

## Описание

Упрощенный "Морской бой" на поле 9x9. Строки и столбцы имеют номера 1-9; у каждой стороны по восемь одноклеточных кораблей.

## Порядок игры

1. Введите время игры в формате `ЧЧММСС`, затем выполните `В/О С/П`. Начальный результат: `10000000`.
2. Введите свой ход `XX`, затем нажмите `С/П`. Результат обозначен как `R`.
3. Если `R = 10000000`, было попадание; вернитесь к пункту 2.
4. Если `R = 1.XX`, был промах; `XX` - координаты выстрела ПМК. Оцените выстрел ПМК: при промахе вернитесь к пункту 2, при попадании перейдите к подготовке в пункте 5.
5. Для подготовки после попадания ПМК выполните `Cx С/П`. Результат снова обозначен как `R`.
6. Если `R = 1.XX`, `XX` - координаты выстрела ПМК; вернитесь к пункту 4.
7. Если `R = 8.EEEEEEE`, игра закончена победой ПМК. Если `R = 8.-------`, игра закончена победой игрока.
8. Для новой игры вернитесь к пункту 1.

## Адаптация для МК-61

Оригинальный рукописный листинг подписан как МК-52, но не использует команды ППЗУ. Командная последовательность сохранена, запись команд приведена к формату МК-61.

Нижняя пометка источника `/0-E/ 98-98-104` сохранена как заполнение адресов `98`-`A4` командой `98`.
