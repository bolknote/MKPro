## Description

Numerical flight-simulator routine for near-planet maneuvering in a near-vacuum environment.

## Notes

- Works as a turn-based manual control model with current height and speed visible.
- You supply thrust and burn duration, then run simulation.
- The game models fuel consumption and overload-based free-fall behavior.
