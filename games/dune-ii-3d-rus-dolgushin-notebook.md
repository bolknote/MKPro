## Описание

Стратегическая игра по мотивам `Dune II`: игрок и CPU собирают спайс, получают кредиты, строят базу и бросают технику на защиту или атаку базы противника.

В тетради программа подписана как "переход с приставки на МК-61".

В ходе игры харвейстеры игрока автоматически собирают и вычисляют спайс, за который выдаются кредиты. Если у базы заканчиваются жизни, она ремонтируется за свой счёт. Если у противника кончаются кредиты, он сдаёт спайс и гасит кредитные долги.

## Подготовка

1. Режим `F АВТ`.
2. Ввести в регистры памяти:

```text
10 хП1
15 хПd
100 хПЕ
63 хПC
88008897 В↑ 851562 K∨ K{x} ВП 7 хП9
0 хП0
80888882 В↑ 856332 K∨ K{x} ВП 7 хПВ
400 хП5       ; кредиты игрока
800 хП6       ; кредиты CPU
50 хП3        ; жизни базы игрока
50 хП2        ; жизни базы CPU
```

## Игра

1. Выполнить `В/О`, `С/П`; на экране будет количество кредитов.
2. В `R0` хранится количество собранного спайса CPU, который потом превращается в кредиты.
3. Ввести действие:

```text
0     ; сбор спайса
-1    ; строительство базы
n     ; количество техники для защиты или атаки базы врага, 1 < n < 16
```

4. После боя количество техники CPU можно посмотреть командой `Пх7`.
5. Сообщение `SELL-82` означает победу: база врага уничтожена.
6. Сообщение `riSE-97` означает атаку базы игрока; нужно ввести количество техники и нажать `С/П`.

## Боевая техника

```text
танки CPU      ; стоят 15 кредитов, очень мощные
бронемашины    ; стоят 10 кредитов, слабее танков CPU
```

Больше 15 штук вводить нельзя. В тетради указано: в игре один этап.
