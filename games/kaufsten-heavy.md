- Author: illi4

## Description

A harder difficulty variant of the Kaufsten puzzle game.

## Note

Use this version when you want a denser challenge than the light variant.

The xbasoft `kaufsten` PMK listing is byte-identical to this file, so the duplicate local variant is not kept separately.
