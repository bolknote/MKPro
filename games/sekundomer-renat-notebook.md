## Назначение

Калибруемый секундомер для МК-61. В регистре `Р9` хранится корректирующий множитель.

## Ввод программы

| Адрес | Команда |
| --- | --- |
| 00 | `Пх9` |
| 01 | `В↑` |
| 02 | `В↑` |
| 03 | `В↑` |
| 04 | `С/П` |
| 05 | `+` |
| 06 | `БП` |
| 07 | `05` |

## Инструкция

1. Ввести начальный множитель: `0,5 хП9`.
2. Нажать `В/О С/П`.
3. По истечении контрольного времени, ориентировочно 60 секунд, нажать `С/П`.
4. Если `X` не равен `60`, изменить корректирующий множитель в `Р9`.

В тетради рядом с листингом указано: `П-х9 - 0,54463`.
