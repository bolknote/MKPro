# Quadratic Equation Solver without Registers (MK61/52)

- Source: [Arbinada — КВУР - квадратное уравнение ...](https://pmk.arbinada.com/category/01167.html)
- Program: [kvur-quadratic-no-register.txt](kvur-quadratic-no-register.txt)

## Description
Registers-free variant of the same quadratic-equation solver for MK-61/52.

This shorter code computes the two roots and supports the complex-root case with the same input format as the register version.
