## Описание

Симулятор управления автомобилем. Игрок задает параметры машины, шаг времени, поворот руля и режим двигателя; программа рассчитывает координаты, скорость и общее время движения.

## Порядок игры

1. Установите угловой режим `градусы`.
2. Введите константы: `F хП1`, `K хП2`, `a хП3`, `m хП4`, `S хП5`.
3. Выполните подготовку `В/О С/П`. Результат `00` означает, что исходные данные приняты.
4. Введите шаг времени `Δt`, с, командой `хП0`.
5. Введите управление: поворот руля `φ`, затем `ПП`; режим двигателя `φдв`, затем `С/П`. Результат - координата `x`, м.
6. Нажмите `↔`, чтобы получить координату `y`, м. Нажмите `Пх6`, чтобы получить скорость `V`, м/с. Нажмите `ПхД`, чтобы получить общее время езды `T`, с.
7. Продолжайте с пункта 4 или 5.

Сообщение `00` означает опрокидывание: резкий поворот на большой скорости. В этом случае `x` и `y` оставляются прежними, скорость зануляется (`V = 0`), а к общему времени прибавляется штраф. В начале `x0 = y0 = V0 = T0 = φ = 0`.

Диапазоны управления: `-1 <= φ <= 1` для руля (влево/вправо) и `-1 <= φдв <= 1` для двигателя (тормоз/разгон).

## Формулы и проверка по коду

На листе приведены расчетные константы:

- `a = hL`
- `F = ³√(a · Nкр²)`
- `K = (10LS/h)²`
- `b = Vкр · m / a`

Здесь `h` - высота машины, `L` - ширина колеи, `S` - база, `Nкр` - максимальная мощность, `m` - масса. По ограничениям и формуле для `b` индекс читается как `кр` - критическая/предельная скорость, а не `пр`.

Адреса `86`-`97` реализуют шаг скорости через силу тяги и сопротивление:

`V' = V + Δt · (F · u - a · V²) / m`

где `u` - текущее значение управления, занесенное программой в рабочую пару `RА/RВ` на этом этапе. Адреса `52`-`71` считают перемещение и изменение курса:

`Δs = Vср · Δt`

`ψ' = ψ + 57 · φ · Δs / S`

Проверка опрокидывания в адресах `43`-`50` сравнивает `(φ · Vср²)²` с `K`; безопасная область соответствует `|φ| · Vср² < sqrt(K) = 10LS/h`.

Ограничения с листа:

- `Δt < |b / (φдв · Vкр² - V²)|`
- `|φдв| < ((b / Δt) + V²) / Vкр²`
- `Δt < S / (V · |φ|)`
- `|φ| < S / (V · Δt)`

## Адаптация для МК-61

Оригинальный рукописный листинг подписан как МК-52, но не использует команды ППЗУ. Командная последовательность сохранена, запись команд приведена к формату МК-61.

Нижняя пометка источника `/0-Д/ 98-98-104` по объяснению автора указывает занятые регистры (`R0`-`RД`) и размер программы: 98 фактически, 98 при округлении до кратности 7, 104 при округлении до кратности 8.
