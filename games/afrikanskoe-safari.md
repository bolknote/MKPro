# Африканское сафари

- Автор: В. Кононов
- Источник: А. Б. Бойко, «Игры с микрокалькулятором», *Математика, кибернетика*, 1987, № 11
- Листинг: рис. 42
- Программа: [afrikanskoe-safari.txt](afrikanskoe-safari.txt)

## Описание

Игра-охота с сетями: игрок вводит координаты охотника, а калькулятор сообщает сведения, по которым нужно определить положение жирафа.
