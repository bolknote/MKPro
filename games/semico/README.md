# Semico MK Programs

Source:

- [Semico MK games](http://mk.semico.ru/mkpr_r9.htm)

This folder contains Semico-hosted listings that fit the current MK-61
emulator model:

- `*.txt` - plain program listings, one command per addressed step.
- `*.md` - Markdown descriptions and setup registers decoded from `*.mkd`
  data files where available.
- `manifest.tsv` - tab-separated index: `program`, `description`, `title`,
  `author`, `source_url`.

Programs: **3**.

Other games on the Semico page target MK-152/MK-161 extensions or exceed the
MK-61 program window, so they are not included here yet.
