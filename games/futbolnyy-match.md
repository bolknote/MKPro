# Футбольный матч

- Source: [Alexander Bulatov GeoCities page, archived](https://web.archive.org/web/20091020020922/http://geocities.com/CapeCanaveral/Hall/5525/Random/soccerru.htm)
- Program: [futbolnyy-match.txt](futbolnyy-match.txt)
- Listing restored from the archived `soc.jpg` image with macOS Vision OCR and manual byte-code check.

The program simulates a football match on an MK-61/MK-52. It outputs the minute of each scored goal and the current score; team strengths are configured before the run.

This is not the same byte-code listing as [futbol.txt](futbol.txt), which is a larger interactive football game.
