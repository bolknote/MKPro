# Lord_BSS MK-52/MK-61 Game Collection

Source index: [Lord_BSS PMK programs](https://lordbss.narod.ru/pmk.html)

This folder contains:

- `pmkNNN.txt` - 254 plain program listings from the source pages.
- `pmkNNN.md` - Markdown descriptions converted from the source pages.
- `manifest.tsv` - generated local index: `program`, `description`, `title`,
  `author`, `source_url`.

The original HTML wrappers, navigation, counters, and scripts are not kept.

Reading guide: [`../../docs/05-reading-lordbss-games.md`](../../docs/05-reading-lordbss-games.md).
