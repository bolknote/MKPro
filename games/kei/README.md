# Tekhnika Molodezhi / KEI originals

Source scans and articles from the «Клуб электронных игр» column in
*Tekhnika Molodezhi*.

This folder contains:

- `*.txt` — plain program listings, one command per addressed step.
- `*.md` — Markdown descriptions converted from the source pages.
- `manifest.tsv` — tab-separated index with program file, description file,
  title, author, and source reference.

Listings are transcribed from scans with macOS Vision OCR and manually checked
against the printed tables.
