# Electronic Fortune-Teller

- Source: *Техника - молодежи*, No. 7, 1985, article «Генератор сюжетов»
- Program: [electronic-fortune-teller.txt](electronic-fortune-teller.txt)
- Listing restored from DjVu scan with macOS Vision OCR and manual table check.

## Description

This is a plot generator based on Stanislaw Lem's «Pocket Computer of the
Beginning Science-Fiction Writer» table. The calculator selects branches in the
printed scheme and outputs the next variant number.

After entering the program, switch to calculation mode and press `В/О`. To start
a new story, set the `Р-Г` switch to `Г`, enter `0`, and press `С/П`. After the
ready stop, enter the number of variants available in the current table node and
press `С/П`. Use the displayed number to follow the printed scheme, then enter
the next variant count and continue with `С/П`.
