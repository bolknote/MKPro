## Описание

Вариант "Танковой баталии", в котором игрок перед выстрелом может посмотреть текущий коэффициент погоды в `R3`. В остальном правила совпадают с вариантом реального боя.

### Подготовка

Рекомендуемые регистры из источника:

- `R0 = 9,81`
- `RD = 50`
- `RE = 50`
- `R1 = 20`, `R2 = 20`
- `R4 = 5`, `R5 = 5`
- `R7 = 4000`, `R8 = 4000`
- `R3` любое число
- `R6 = 0`
- `RB = 250`

Переключатель `Р-Г` поставьте в градусы.

### Запуск

Нажмите `БП 98 С/П`. Затем введите `1` для хода танка N1 или `2` для хода танка N2. Для выстрела введите `угол`, `↑`, `скорость`, `С/П`.
