- Series: Kon-Tiki / Путь к Земле

## Description

This short utility computes planetary gravity from radius and an empirical constant.
In the issue text it is used by the landing games and the `Guess Gravity` activity to create
custom test worlds with user-selected planets.

The listing is restored from the published command sequence in the source article.
