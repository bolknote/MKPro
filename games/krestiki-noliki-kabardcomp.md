# Крестики-нолики

- Источник: [kabardcomp.narod.ru/Mgames.txt](https://kabardcomp.narod.ru/Mgames.txt)
- Программа: [krestiki-noliki-kabardcomp.txt](krestiki-noliki-kabardcomp.txt)

## Описание

Короткий вариант игры "крестики-нолики" для ПМК.

Установите переключатель `Р-Г` в положение `Р` и нажмите `С/П`. На индикаторе появится номер клетки, куда сделал ход калькулятор. Введите номер клетки для своего хода и нажмите `С/П`.

Если калькулятор выиграл, на индикаторе будет `77`. Если партия закончилась вничью, будет `0`.

## Нумерация клеток

```text
   |   |
 1 | 2 | 3
   |   |
---+---+---
   |   |
 8 | 9 | 4
   |   |
---+---+---
   |   |
 7 | 6 | 5
   |   |
```

## Примечание

Это отдельный короткий вариант. Он не совпадает по bytecode с уже существующими локальными вариантами "Крестики-нолики".
