# Рисование

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 216
- Программа: [risovanie-romanovsky-1989.txt](risovanie-romanovsky-1989.txt)

## Описание

Учебная графическая программа: последовательно выдает пары координат для рисования фигур по заданным параметрам.

Координаты нужно переносить на бумагу после остановов `С/П`.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
