# Markov Machine

- Source: [Arbinada — Машина Маркова (52, 61)](https://pmk.arbinada.com/category/00960.html)
- Program: [markov-machine.txt](markov-machine.txt)

## Description
A Markov rewrite machine simulator for MK-61.

It keeps a rule table in registers and repeatedly applies substitution rules to a work string, stopping when no rules match or a terminating rule is encountered.
