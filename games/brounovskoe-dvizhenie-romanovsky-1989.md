# Броуновское движение

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 212
- Программа: [brounovskoe-dvizhenie-romanovsky-1989.txt](brounovskoe-dvizhenie-romanovsky-1989.txt)

## Описание

Учебная имитация броуновского движения: программа выдает координаты случайных шагов для построения траектории.

В листинг включена общая подпрограмма генератора случайных чисел с адресов 75-96.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
