# Мультфильм на индикаторе

- Автор: К. Котин
- Источник: "Наука и жизнь", N6 за 1989 год, стр. 106
- Программа: [multfilm-na-indikatore.txt](multfilm-na-indikatore.txt)

## Описание

Демонстрационная программа, которая выводит на индикаторе несколько кадров "космического" мультфильма.

### Подготовка

После ввода программы занесите кадры в регистры:

- `10888888 В↑ 13747737 К∨ K{x} хП1 FL0`
- `13747237 К∧ K{x} хП2 FL0`
- `13737732 К∨ K{x} хП3 FL0`
- `13774772 К∨ K{x} хП4 FL0`
- `13774727 К∨ K{x} хП5 FL0`
- `13774277 К∨ K{x} хП6 FL0`
- `13724777 К∨ K{x} хП7 FL0`
- `13724777 К∨ K{x} хП8`
- `3 /-/ хП9`

Для запуска нажмите `В/О С/П`; для повторного показа нажимайте `С/П`.
