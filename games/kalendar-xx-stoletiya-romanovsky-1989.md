# Календарь XX столетия

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 206-208
- Программа: [kalendar-xx-stoletiya-romanovsky-1989.txt](kalendar-xx-stoletiya-romanovsky-1989.txt)

## Описание

Календарная программа для XX века: определяет день недели и считает интервалы между датами.

Раздел БК-0010 из той же книги не включен, потому что это не ПМК-листинг.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
