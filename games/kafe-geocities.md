# Кафе (GeoCities)

- Author: В. Алексеев, по идеям Л. Самутина
- Source: [Alexander Bulatov GeoCities page, archived](https://web.archive.org/web/20091020052223/http://geocities.com/CapeCanaveral/Hall/5525/Cafe/Caferu.htm)
- Original publication: *Техника - молодежи*, No. 5, 1988
- Program: [kafe-geocities.txt](kafe-geocities.txt)
- Listing restored from the archived `cafe.jpg` image with macOS Vision OCR and manual byte-code check.

Economic cafe simulator for MK-61/MK-54. The player allocates money to food, advertising, and service expenses; the program computes visitors and income.

This GeoCities listing has a different byte-code sequence from the existing `cooperative-cafe*` and `kooperativnoe-kafe*` variants, so it is kept separately.
