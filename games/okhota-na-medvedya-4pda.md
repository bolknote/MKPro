## Описание

Доработанная версия игры [Охота на медведя](okhota-na-medvedya.md). Основной алгоритм сохранен, но координаты медведя стали прямыми, а не инверсными. После отображения числа для стрельбы оно удаляется из стека; вместо него можно видеть число жизней медведя и оставшиеся патроны.

После удачного или неудачного выстрела больше не нужен дополнительный `С/П`: при попадании отображается `0`, при промахе - неугаданное число, и затем можно продолжать ходить.

### Подготовка

Регистры:

- `R0` - не требуется инициализация
- `R1` - не требуется инициализация
- `R2 = 10`
- `R3 = 3`
- `R4 = 0`
- `R5` - не используется
- `R6 = -5`
- `R7 = 1`
- `R8` - не требуется инициализация
- `R9 = 5`
- `RA = 43`
- `RB` - случайное число от 0 до 1, не ноль
- `RC = 55555555`; можно оставить авторское `85555555`
- `RD = 60`
- `RE = 98`

### Настройки

- Радиус нападения зверя `N1 = 2`: шаг `13` содержит `3`, шаг `19` содержит `4`.
- Максимальная скорость зверя `N2 = 2`: шаг `A3` содержит `2`, шаг `A0` содержит `5`.
- Усталость охотника `N4 = 9`: шаг `65` содержит `9`. Чтобы отключить усталость, замените шаги `64`-`68` на `КНОП`.

### Запуск

Задавайте ход как `смещение Y`, `В↑`, `смещение X`. Если привычнее старый порядок координат, в первой строке программы можно поменять местами `R6` и `R7`.
