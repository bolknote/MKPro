## Description

Игра между ПМК и игроком на циклически замкнутом поле 4x4. Нужно поставить четыре знака в ряд. Учитываются 4 вертикали, 4 горизонтали и 8 сквозных диагоналей, поэтому выигрышная комбинация может сложиться по одной из 16 линий.

Координаты клетки задаются парой `X:Y`, где `X` - горизонталь, `Y` - вертикаль, оба значения от 1 до 4:

|   |   |   |   |
|---|---|---|---|
| 1:4 | 2:4 | 3:4 | 4:4 |
| 1:3 | 2:3 | 3:3 | 4:3 |
| 1:2 | 2:2 | 3:2 | 4:2 |
| 1:1 | 2:1 | 3:1 | 4:1 |

### Начальная настройка

Однократно, перед серией игр, введите константы:

- RA = `Г.       -02` (`55 В↑ 88 K∨ K{x} ВП /-/ 1`)
- RB = `88888834`
- RC = `2.2600029-01`
- RD = `4.1200076-01`

Переключатель `Р-ГРД-Г` установите в положение `Г`.

Перед каждой игрой:

- R4...R7 = `44444.4`
- R9 = `0`
- RA должно быть положительным
- `В/О С/П`

Если первым ходит ПМК: сделайте RA отрицательным, введите в R0 число больше 0 и меньше 99, затем `Б/П 07 С/П`.

### Ход игры

Игрок вводит ход так: `X ПП Y С/П`.

Ответ ПМК:

- RX = `X`, RY = `Y` - ход ПМК, после него игрок вводит следующий ход.
- RX = `-99999999` - игрок выбрал уже занятую клетку; нужно повторить ход.
- RX = `8.       -0n`, где `n` от 1 до 4 - ПМК выиграл; победный ход находится в R1 (`X`) и R2 (`Y`).
- RX = `0` - свободных ходов больше нет, ничья.

ПМК автоматически нормализует координаты: дробная часть отбрасывается, затем число приводится по модулю 4 к диапазону 1...4. Например, `-1` становится `3`, `0` становится `4`, `9` становится `1`.

### Используемые регистры

- R0 - косвенная адресация / оценка текущей позиции
- R1, R2 - координаты текущей позиции
- R3, R8 - координаты наилучшей позиции или выводимое сообщение
- R4...R7 - накопленные суммы знаков по 16 линиям
- R9 - занятые клетки
- RA - флаг игрока и адрес перехода
- RB...RD - константы, служащие также адресами перехода
- RE - адрес перехода
