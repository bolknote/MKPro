## Описание

Лабиринтная игра про спелеолога, который ищет саркофаг в африканской пещере. Игрок видит только часть лабиринта, расходует заряд фонаря и должен добраться до сокровища.

## Предыстория

Африка, 1949 год. Солнце безжалостно палит голову. Отряд спелеологов идёт через пустыню на караване и к вечеру должен успеть прийти к скалистому массиву, цели похода.

Когда солнце почти спустилось за горизонт, путники обнаружили на горизонте горы. У входа в пещеру, где по легендам должен быть саркофаг, установили специальные приспособления для входа в глубину. Спуск начался.

## Подготовка

В режиме `F АВТ` ввести начальные значения:

```text
11111111 хП9
1 хП1
1 хП4
58 хП7
63 хП8
15 хПA
15 хПC
0 хП6
150 хП0
500 хП2
88008 В↑ 83056 K∨ K{x} ВП 4 хП3
8.81 хП5
81507011 KИНВ хПd
8880088 В↑ 8200132 K∨ K{x} ВП 5 хПE
```

`150 хП0` задаёт запас электричества в аккумуляторе фонаря. `500 хП2` задаёт координату цели. Значение `хП5` создаёт лабиринт: в источнике указано число не меньше `7` и не больше `150`, можно дробное, например `8.81`.

Переключатель `Р-ГРД-Г` задаёт сложность. В режиме `Р` лабиринт повышенной сложности, в режиме `Г` лёгкий. `ГРД` не ставить.

Для начала игры нажать `В/О`, `С/П`.

## Экран

На экране показывается часть лабиринта.

| Символ | Значение |
| --- | --- |
| `-` | спелеолог |
| `E` | стена |
| пробел | коридор |
| `8` | сокровище, саркофаг |

## Управление

| Ввод | Действие |
| --- | --- |
| `4`, `С/П` | влево |
| `6`, `С/П` | вправо |
| `8`, `С/П` | вверх |
| `2`, `С/П` | вниз |

## Сообщения

| Сообщение | Значение |
| --- | --- |
| `LOSE` | аккумулятор фонаря закончился, конец игры |
| `BOIL-` | на пути стена, тупик |
| `8E-8EE` | победа, саркофаг найден |

## Примечание к источнику

В основном листинге по адресу `60` стоит `5`. В источнике под таблицей указано, что по адресу `60` можно поставить `3`, тогда спелеолог будет отображаться как `C`.
