# Задача о разорванной книге

- Source: [Alexander Bulatov GeoCities page, archived](https://web.archive.org/web/20010422032243/http://www.geocities.com/CapeCanaveral/Hall/5525/Tornbook/Tornbru.htm)
- Program: [razorvannaya-kniga.txt](razorvannaya-kniga.txt)
- Listing restored from the archived `tornb.jpg` image with macOS Vision OCR and manual byte-code check.

Mathematical puzzle program for MK-61. Given a book with `N` pages and a torn-out consecutive block, the program searches for possible values consistent with the remaining page sums.

This is not a game, but it is a calculator program from the same GeoCities archive.
