## Что означает "положение магараджи"

Это текущая клетка единственной белой фигуры после хода игрока. В шахматном варианте "Магараджа и сипаи" белые играют одной фигурой, магараджей; она ходит как ферзь или как конь. Черные играют обычной полной армией.

Поэтому инструкция тетради "следующие ходы делать положением магараджи и С/П" означает: после остановки калькулятора сделать ход магараджей на доске, ввести код клетки, где теперь стоит магараджа, и нажать `С/П`.

Справка по варианту: [Maharajah and the Sepoys](https://en.wikipedia.org/wiki/Maharajah_and_the_Sepoys).

## Начальная загрузка регистров

После `F АВТ` ввести:

| Регистр | Значение |
| --- | --- |
| `Р1` | `454` |
| `Р2` | `443` |
| `Р3` | `44` |
| `Р4` | `873` |
| `Р5` | `883` |
| `Р6` | `83` |
| `Р7` | `763` |
| `Р8` | `123` |
| `Р9` | `113` |
| `РA` | `14` |
| `РB` | `233` |
| `РC` | `54` |
| `Рd` | `375` |

Затем нажать `В/О С/П`.

## Инструкция

1. Ввести программу после `F ПРГ`.
2. Перейти в `F АВТ`.
3. Загрузить регистры по таблице выше.
4. Нажать `В/О С/П`.
5. После каждого хода калькулятора ввести положение магараджи и нажать `С/П`.

## Примечания по записи

В тетради команды `Пх0`, `ПхC`, `Пхd` записаны как `ИП0`, `ИПС`, `ИПд`.

Команда `КПх0` записана как `КИП0`.
