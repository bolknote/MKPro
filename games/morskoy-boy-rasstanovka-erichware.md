- Источник: [ErichWare, программы для МК-52](https://erichware.com/sovkalk/texty/promk52.htm), лист [52morboi.gif](https://erichware.com/sovkalk/texty/mk52/52morboi.gif)
- Автор: [Эрих Генрихович Гаузер](https://erichware.vgd.name/), [Баку](https://erichware.org/show/baku/bakufoto.htm)
- Программа: [morskoy-boy-rasstanovka-erichware.txt](morskoy-boy-rasstanovka-erichware.txt)
- Связанная программа: [morskoy-boy-igra-erichware.txt](morskoy-boy-igra-erichware.txt)

## Описание

Подготовительная программа для игры "Морской бой": расставляет корабли ЭВМ и сохраняет состояние в регистрах. После ее выполнения нужно сразу загрузить программу "Игра", не очищая регистры.

Игра ведется на поле 7×7 с номерами 1-7, с каждой стороны участвуют по 8 одноклеточных кораблей.

## Порядок запуска

1. Введите начальное число `0,ДДЧЧММСС`, где `ДД` - дата игры, `ЧЧ` - час, `ММ` - минуты, `СС` - секунды.
2. Выполните `В↑ В/О С/П`.
3. Результат должен быть `0`.
4. Для продолжения загрузите программу [morskoy-boy-igra-erichware.txt](morskoy-boy-igra-erichware.txt), не очищая регистры.

## Адаптация для МК-61

Оригинальный рукописный листинг подписан как МК-52, но в этой части не используются команды ППЗУ. Командная последовательность сохранена, запись команд приведена к формату МК-61.

Программы "Расстановка" и "Игра" вместе занимают 147 команд, поэтому не помещаются в память МК-61 как один листинг. Для МК-61 они сохраняются как связанная пара: сначала выполняется "Расстановка", затем поверх нее загружается "Игра" с сохранением регистров.

Нижняя пометка источника `/0-Е/ 84-84-88` по объяснению автора указывает занятые регистры (`R0`-`RЕ`) и размер программы: 84 фактически, 84 при округлении до кратности 7, 88 при округлении до кратности 8.
