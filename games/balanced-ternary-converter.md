# Balanced Ternary Converter

- Source: [Habr — МК-61: история, эмуляция, устройство](https://habr.com/ru/articles/505612/)
- Program: [balanced-ternary-converter.txt](balanced-ternary-converter.txt)

## Description

Converts an input decimal number to balanced ternary notation.

The result is displayed as decimal digits where `7`, `8`, and `9` represent
`-1`, `0`, and `1` respectively. For example, input `123` produces `977778`,
which corresponds to `+----0` in balanced ternary.

## Setup

Enter the number in `X`, then run with `В/О С/П`.
