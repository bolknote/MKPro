# Магический квадрат

- Автор: М. Вахтеров (Москва)
- Источник: А. Б. Бойко, «Игры с микрокалькулятором», *Математика, кибернетика*, 1987, № 11
- Листинг: рис. 3
- Программа: [magicheskiy-kvadrat.txt](magicheskiy-kvadrat.txt)

## Описание

Головоломка на восстановление магического квадрата 3x3. Листинг восстановлен по таблице кодов из брошюры; команды сверены по шестнадцатеричным кодам.
