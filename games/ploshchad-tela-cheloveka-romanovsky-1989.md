# Площадь тела человека

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 203
- Программа: [ploshchad-tela-cheloveka-romanovsky-1989.txt](ploshchad-tela-cheloveka-romanovsky-1989.txt)

## Описание

Учебная расчетная программа: по росту и массе человека вычисляет площадь поверхности тела.

Перед запуском заносятся постоянные из книги; результат выдается после `В/О` `С/П`.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
