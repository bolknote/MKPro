## Описание

Игра-симулятор ставок на скачках. В заезде участвуют девять лошадей. Игрок выбирает лошадь, делает ставку, следит за выбывающими участниками и пытается не обанкротиться.

## Подготовка

В режиме `F АВТ` ввести:

```text
0.1 хП1
0.1 хП2
0.1 хП3
8.5555555 KИНВ хПB
0.1 хП4
0.1 хП5
0.1 хП6
0.1 хП7
0.1 хП8
0.1 хП9
4 хП0
1000 хПA
```

`хП0` задаёт количество кругов гонки, в источнике указан пример `4`. `хПA` задаёт деньги игрока, в источнике указан пример `1000`.

Для начала нажать `В/О`, `С/П`.

## Ход игры

На экране появляется `8,---`; нажмите `↔`, чтобы увидеть количество денег. Затем ввести ставку в `хПd`, номер лошади, на которую ставите, и нажать `С/П`.

Всего в заезде участвуют `9` лошадей.

## Вывод гонки

| Индикатор | Значение |
| --- | --- |
| целое число | номер лошади, которая вышла из гонки |
| `0` | выигравшая лошадь |
| `8-----` | следующая гонка |
| `ЕГГ0Г` | игрок банкрот |

Если ставка выиграла, деньги прибавляются. Если ставка проиграла, деньги списываются. После результата нажать `С/П`.

Характеристики лошадей можно посмотреть через регистры: например, `Пх5` показывает характеристику пятой лошади. Чем больше число характеристики, тем выше шанс выигрыша.

## Примечания к листингу

В источнике по адресу `29` красным указано `3 or 2 or 4`, при этом `2` подчёркнуто; в листинг внесён основной подчёркнутый вариант `2`. В подписи указано: чем больше число по адресу `29`, тем реже будут вылетать лошади из гонки.
