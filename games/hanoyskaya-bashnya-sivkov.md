# Ханойская башня, вариант Ю. Сивкова

- Автор: Ю. Сивков
- Источник: "Наука и жизнь", N12 за 1989 год, стр. 75-76
- Программа: [hanoyskaya-bashnya-sivkov.txt](hanoyskaya-bashnya-sivkov.txt)

## Описание

Программа решает головоломку "Ханойская башня" для пирамиды до восьми кружков. В мантиссе показано расположение кружков на стержнях, а в знаках порядка - очередной ход в формате "откуда-куда".

### Подготовка

После ввода программы занесите постоянные:

- `10 хП0`
- `27 хПД`
- `41 хПЕ`

В регистр `X` занесите номера кружков: например, `1.23` для трех кружков или `1.2345678` для восьми. Точка после единицы обязательна.

### Запуск

Нажмите `В/О С/П`. Первый ход появляется через 10-15 секунд. Для следующих ходов нажимайте `С/П`. После полного переноса пирамиды появляется надпись `ВСЕ`.
