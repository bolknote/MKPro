# Устный счет

- Автор: коллектив авторов
- Источник: Кибернетика. Микрокалькуляторы в играх и задачах, 1986, с. 16-18
- Программа: [ustnyy-schet-kibernetika-1986.txt](ustnyy-schet-kibernetika-1986.txt)

## Описание

Тренажер устного счета: ПМК выдает арифметические задания, игрок вводит результат и получает оценку.

Сложность повышается по мере правильных ответов.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
