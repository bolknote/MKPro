# Игра на дорожке

- Автор: коллектив авторов
- Источник: Кибернетика. Микрокалькуляторы в играх и задачах, 1986, с. 145
- Программа: [igra-na-dorozhke-kibernetika-1986.txt](igra-na-dorozhke-kibernetika-1986.txt)

## Описание

Учебная программа из главы о теории игр для игры на дорожке.

ПМК анализирует позицию и помогает выбрать ход по описанной в книге стратегии.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
