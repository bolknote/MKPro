# Угадай функцию

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 211-212
- Программа: [ugaday-funktsiyu-romanovsky-1989.txt](ugaday-funktsiyu-romanovsky-1989.txt)

## Описание

Игра на распознавание функции: ПМК выбирает одну из нескольких функций, а игрок по пробным значениям должен определить какую.

В листинг включена общая подпрограмма генератора случайных чисел с адресов 75-96.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
