## Описание

Многоуровневая игра по мотивам `Dune`. Игрок управляет харвейстером, собирает спайс-минерал, следит за приборами и избегает песчаных червей.

В тетради указано: игра написана 5 октября 2000 года. Подпись в источнике: `LEX..!X`.

Для начала предварительно выбрать маршрут, затем нажать:


Первоначально:

```text

```

## Правила

Вы управляете харвейстером, и судьба базы в ваших руках.

Для обнаружения спайса у харвейстера под фюзеляжем установлен `S-SENSOR`. Он работает по 6-балльной системе: чем выше балл, тем больше в почве спайс-минерала.

Собирать спайс мешают песчаные черви. Если песчаный червь приблизится меньше чем на `1` км, он проглотит харвейстер: `GAME OVER`.

Для обнаружения червей у харвейстера есть `E-SENSOR`. Он всё время фиксирует передвижение червей и показывает расстояние от ближайшего червя до харвейстера в километрах.

Набрав побольше спайса, отвезите его на базу. При пересечении границы базы спайс автоматически выгружается.

Чтобы перейти на следующий уровень, нужно собрать определённое количество спайса, указанное в `РА`. После прохождения уровня на экране появляется:

```text
-0C-300
```

После этого ввести:

```text
300 хПA
```

и продолжить игру на другом уровне. Чем дальше зайдёте, тем меньше спайса будет в почве.

## Управление навигационной рубкой

```text
1      ; поворот влево на 45°
3      ; поворот вправо на 45°
0      ; руль в среднем положении
/-/ 1  ; переход в рубку управления
Пх1    ; координата харвейстера по X
Пх2    ; координата харвейстера по Y
```

Чтобы узнать курс харвейстера:

```text
Пх9
ПхC
÷
```

Целое число результата — курс.

## Рубка управления

Символ рубки управления: `---`.

```text
N В↑ 0 ; включение монитора, где N - номер монитора
1      ; насос ON
2      ; насос OFF
3      ; стоп двигатель
4      ; максимальная скорость
/-/ 1  ; навигационная рубка
```

## Мониторы

```text
N3 - E-SENSOR
N4 - S-SENSOR
N5 - количество спайса, загруженного в харвейстер
N6 - скорость харвейстера
N9 - курс харвейстера
```

После включения монитора его информация будет передаваться на главный монитор навигационной рубки.

Если за вами гонится червь, убегайте от него на базу.

Когда наберёте `100` тонн спайс-минерала, выключите насос и езжайте на базу.

Совет из тетради: не сбивайтесь с курса и помните, в каком положении находится руль.

После каждой игры восстановить:

```text
7 хП0
0 хП1
0 хП2
32 хП3
Сх хП5
300 хПA
0 хПd
```

## Примечание к листингу

По адресу `47` в тетради стоит `K√`. По пояснению автора, эта команда намеренно вызывает `ERROR`; это самый короткий способ получить сообщение об ошибке. В листинге сохранено `K√`.
