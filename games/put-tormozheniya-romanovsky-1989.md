# Путь торможения

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 204
- Программа: [put-tormozheniya-romanovsky-1989.txt](put-tormozheniya-romanovsky-1989.txt)

## Описание

Учебная программа для оценки тормозного пути автомобиля по скорости.

Использует табличные коэффициенты из книги и останавливается с рассчитанным расстоянием.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
