# Решение шахматной задачи Ллойда

Полный листинг: [lloyd-queen-c5-renat-notebook.txt](lloyd-queen-c5-renat-notebook.txt)

Игра ферзем `C5`. Автор неизвестен.

Источник: тетрадь Рената Менькова "Мои игры МК-61 (Тетрадь 2)", `Рисунок (13).jpg`.

## Инструкция

1. Нажать `В/О С/П`.
2. Для высвечивания первого хода белых используется `44` (`Д4`).
3. Затем ввести код поля, занимаемого ферзем при ходе черных, в формате `xy`.
4. Нажимать только `С/П` для высвечивания следующего хода белых.

Обозначение `код поля <-> (xy)` в тетради означает соответствие кода поля координатам `xy`, а не нажатие клавиши `x<->y`.

## Примечания по тетради

У команды `30 С/П` стоит пометка `F ПРГ`.

После `БП 00` стоит пометка `F АВТ` и нарисована загнутая стрелка вниз.
