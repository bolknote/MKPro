# Числа по кругу

- Авторы: И. Д. Данилов, Г. В. Славин
- Источник: И. Д. Данилов, Г. В. Славин, "5 вечеров с микрокалькулятором", с. 32
- Программа: [chisla-po-krugu.txt](chisla-po-krugu.txt)

## Описание

Модель циклического стека из пяти чисел. Игра показывает, как меняется стек после операций, выбранных по числовому коду.

Запуск: `БП 67 С/П`; начальное состояние на индикаторе - `11111`.

Введите код операции и нажмите `С/П`; на индикаторе появится новое положение чисел в стеке. Коды: `1` - `+`, `2` - `-`, `3` - `×`, `4` - `Fx²`, `5` - `F↻`, `6` - `↔`, `7` - `FВx`. Для ввода новой цифры используйте `цифра В↑ 8 С/П`.

## Примечание к листингу

Листинг нормализован из старой формы записи команд для Б3-34.
