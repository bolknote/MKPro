# Одномерный кубик Рубика

- Автор: Г. Славин (Тарту)
- Источник: А. Б. Бойко, «Игры с микрокалькулятором», *Математика, кибернетика*, 1987, № 11
- Листинг: рис. 6
- Программа: [odnomernyy-kubik-rubika.txt](odnomernyy-kubik-rubika.txt)

## Описание

Одномерная перестановочная головоломка: из случайного порядка цифр нужно получить упорядоченную последовательность, двигая элементы по правилам игры. Листинг восстановлен по таблице кодов.
