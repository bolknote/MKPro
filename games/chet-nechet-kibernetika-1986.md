# Чет-нечет

- Автор: коллектив авторов
- Источник: Кибернетика. Микрокалькуляторы в играх и задачах, 1986, с. 154
- Программа: [chet-nechet-kibernetika-1986.txt](chet-nechet-kibernetika-1986.txt)

## Описание

Игра «чет-нечет» из раздела о самообучающихся игровых программах.

ПМК накапливает статистику ходов игрока и пытается выбирать стратегию по обнаруженным закономерностям.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
