# Угадай число

- Автор: коллектив авторов
- Источник: Кибернетика. Микрокалькуляторы в играх и задачах, 1986, с. 82, 106
- Программа: [ugaday-chislo-kibernetika-1986.txt](ugaday-chislo-kibernetika-1986.txt)

## Описание

Учебная версия игры «Угадай число» из главы о программировании.

ПМК ведет диалог с игроком, использует генератор случайных чисел и сужает диапазон поиска.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
