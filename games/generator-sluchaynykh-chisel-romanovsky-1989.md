# Генератор случайных чисел

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 209
- Программа: [generator-sluchaynykh-chisel-romanovsky-1989.txt](generator-sluchaynykh-chisel-romanovsky-1989.txt)

## Описание

Вспомогательная программа `RND(A,B)`: возвращает псевдослучайное целое число из заданного диапазона.

По источнику этот фрагмент вводится с адреса 70; в карточках игр ниже используется его подпрограмма с адресов 75-96.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
