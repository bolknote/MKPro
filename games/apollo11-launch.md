# Apollo 11 Launch from Earth (Refined)

- Source: [Arbinada — Аполлон 11. Старт с Земли (МК61)](https://pmk.arbinada.com/category/01617.html)
- Program: [apollo11-launch.txt](apollo11-launch.txt)

## Description
Refined MK-61 ascent model for Apollo-style flight with combined atmospheric scale and aerodynamic parameter in a single register.

The routine is intended to run with standard start conditions and provides stabilized thrust vector/autopilot interaction for practical first-stage ascent behavior.
