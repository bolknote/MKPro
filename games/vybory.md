# Выборы

- Автор: Виктор Распопов
- Источник: *Пионерия/Одноклассник*, №2, 1990, mirrored at [arbinada.ddns.net](http://arbinada.ddns.net/pmk-files/magazines/%D0%9F%D0%B8%D0%BE%D0%BD%D0%B5%D1%80%D0%B8%D1%8F%201990-91%20%D0%9F%D0%9C%D0%9A.zip)
- Программа: [vybory.txt](vybory.txt)

## Описание

Программа подсчитывает результаты голосования с помощью калькулятора. После ввода программы включите калькулятор, нажмите `F` `ВП`, затем `F` `/-/` `В/О`.

Голосование ведется по номерам кандидатов. Результаты накапливаются в регистрах `1`, `2`, ..., `C`.

## Примечание

Печатный листинг был набран в обозначениях другой машины; здесь команды записи и чтения регистров адаптированы к МК-61.
