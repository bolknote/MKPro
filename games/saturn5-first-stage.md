# Saturn V First Stage (MK61)

- Source: [Arbinada — Первая ступень САТУРН-5 на МК61](https://pmk.arbinada.com/category/01619.html)
- Program: [saturn5-first-stage.txt](saturn5-first-stage.txt)

## Description
Moon-capable launch-vehicle simulation for first-stage flight on Saturn-like rockets.

Uses MK-61 registers for altitude/velocity state and supports thrust, drag, and gravity handling during powered flight; intended for integration into multi-stage mission profiles.
