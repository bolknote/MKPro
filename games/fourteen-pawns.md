# Fourteen Pawns

- Source: [Arbinada — 14 пешек (61, 52)](https://pmk.arbinada.com/category/01085.html)
- Program: [fourteen-pawns.txt](fourteen-pawns.txt)

## Description
A board-game engine for the 4x7 variant with 14 pawns that uses logical instructions from MK-61/52.

Store the initial position in registers (start setup for both sides is in the article), input moves with the required coordinates, and let the program evaluate legal moves and game end conditions.
