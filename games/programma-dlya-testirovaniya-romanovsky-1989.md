# Программа для тестирования

- Автор: Т. Б. Романовский
- Источник: Т. Б. Романовский, Микрокалькуляторы в рассказах и играх, Киев: Радянська школа, 1989, с. 217-218
- Программа: [programma-dlya-testirovaniya-romanovsky-1989.txt](programma-dlya-testirovaniya-romanovsky-1989.txt)

## Описание

Учебная программа для проверки тестов с ответами да/нет.

Она кодирует эталон, задает вопросы и выводит количество ответов, совпавших с эталоном.

## Примечание к листингу

Листинг переписан из старой формы записи команд для Б3-34 в современную запись МК-61: `ПR`/`ИПR` заменены на `хПR`/`ПхR`, `КПR`/`КИПR` - на `КхПR`/`КПхR`.
