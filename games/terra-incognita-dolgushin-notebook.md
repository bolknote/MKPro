## Описание

Космический шутер на поверхности планеты `ZANOR`. Игрок управляет истребителем типа `X-wing`, перемещается влево и вправо и стреляет по военным объектам Империи.

## Предыстория

Космическим Освободительным Центром была обнаружена одна из главных баз Империи на планете `ZANOR`. К. О. Ц снарядил войско, состоящее из двадцати крейсеров с истребителями типа `X-wing`.

Подлетая к планете `ZANOR`, было уничтожено много кораблей, но истребители К. О. Ц проникли на планету. Их задача заключалась в уничтожении вооружения на планете. Игрок выполняет роль одного из истребителей: уничтожайте всё, что попадётся.

## Подготовка

В режиме `F АВТ` ввести в регистры:

```text
38 хП0
100 хПE
87500057 KИНВ хП1
87050507 KИНВ хП2
87005007 KИНВ хП3
800055 KИНВ хП4
800077 KИНВ хП5
8888008 В↑ 8772002 K∨ хП6
88880088 В↑ 84270072 K∨ хП7
88880088 В↑ 82470073 K∨ хП8
8888008 В↑ 8774003 K∨ хП9
```

В источнике рядом с `100 хПE` указано: «пушки».

Для начала игры нажать `В/О`, `С/П`.

## Управление

Переключатель `Р-ГРД-Г` используется как управление.

| Положение переключателя | Действие |
| --- | --- |
| `Р` | влево |
| `Г` | вправо |
| `ГРД` | стрельба |

## Обозначения

| Символ | Значение |
| --- | --- |
| `E` | жилой дом |
| `Г` | радарная башня |
| `C` | склад оружия и космопорт |
| `L` | полицейский участок |
| `-` | танк |
| `9` | лазерная установка |
| `8` | казарма, вооружённая |
| `7` | коммерческий центр военной техники |
| `6` | ракетная установка |

## Сообщения

| Сообщение | Значение |
| --- | --- |
| `ЕГГ0Г` | вы уничтожены |
| `COOL` | защита планеты уничтожена, планета обезоружена |

## Примечания к листингу

В листинге адреса `07` и `20` написаны как `K[]`. По логике программы оба места восстанавливаются как `K[x]`: сразу перед ними генерируется случайное число, после чего нужна целая часть для выбора координат и объекта.
