# Ханойская башня, вариант В. Хмелюка

- Автор: В. Хмелюк
- Источник: "Наука и жизнь", N12 за 1989 год, стр. 76-77
- Программа: [hanoyskaya-bashnya-khmelyuk.txt](hanoyskaya-bashnya-khmelyuk.txt)

## Описание

Рекурсивная реализация головоломки "Ханойская башня" для МК-61. Программа использует стек адресов подпрограмм и переносит пирамиду с числом кружков `2 <= N <= 7`.

### Подготовка

После ввода программы занесите постоянные:

- `56 хПА`
- `60 хПВ`
- `72 хПС`

В регистр `X` введите число кружков `N`.

### Запуск

Нажмите `В/О С/П`. Первый ход вычисляется около 10 секунд, последующие - после нажатия `С/П` за 3-8 секунд. После переноса всех кружков на МК-61 появляется надпись `ВСЕ`.

Для Б3-34 в источнике указана замена окончания программы: `80.84 81.3 82.F 1/x 83.С/П 84.В/О`.
