# Easter Date (Orthodox, Primary)

- Source: [Arbinada — Вычисление даты Пасхи (61/52)](https://pmk.arbinada.com/category/00935.html)
- Program: [easter-date-passover.txt](easter-date-passover.txt)

## Description
Computes the Orthodox Easter date for the input year. Enter the year into register `X`; after execution the month appears in `X` and the day in `Y`.

The program is suitable for MK-61/52 style workflow and can be used with any year in the allowed range of the original routine.
