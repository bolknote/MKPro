# Vostok-1 Atmosphere Program

- Source: [Arbinada — ВОСТОК-1 на МК61](https://pmk.arbinada.com/category/01621.html)
- Program: [vostok-1-atmosphere.txt](vostok-1-atmosphere.txt)

## Description
Atmospheric-segment routine for Vostok-1 mission simulations (second and third stages, orbital correction, re-entry and parachute descent).

This program uses the same command style as the carrier routine but with a lighter flight model for later flight phases and atmospheric braking behavior.
